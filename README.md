A simple and interactive desktop application built using Python and Tkinter that helps users study using custom flashcard sets loaded from a JSON data store file.

## Features
* *Randomized Card Order:* Pulls flashcards dynamically using Python's random package module.
* *Modern Style Layout:* Uses an enhanced blue/green interactive palette design layout.
* *External JSON Storage:* Reads question data dynamically from an independent flashcards.json file configuration source.

## How to Run
1. Ensure Python 3 is installed on your operating system environment.
2. Download both repository project files into the same directory folder.
3. Launch your command prompt or terminal panel inside the repository directory.
4. Execute the following start script application launch trigger command:
   bash
   python main.py
